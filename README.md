Send Dolibarr logs to a Sentry server
=====================================

![Sentry logo](img/sentry.png)

https://sentry.io

For [Dolibarr](https://dolibarr.org) >= 5.0.x.
Sentry support was included in Dolibarr core 3.9 and is broken in 4.x.
